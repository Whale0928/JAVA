## 제네릭 <Generic>

타입의 자료형등을 실행기 시 예외 처리하도록 컴파일할 때 점검할 수 있도록 한것.

#### 제네릭 타입의 이름 명명 규칙

제네릭 타입 선언 할 때 클래스 선언 시 꺽쇠 안에 어느 단어가 들어가도 상관은 없지만 "기본적으로 자바의 규칙은 있다."

- E : 요소 (Element , 컬렉션에서 하나하나를 지칭할때 주로 사용)
- K : 키
- N : 숫자
- T : 타입
- V : 값
- S,U,V : 두 번째 , 세 번째 , 네 번쩨에 선언된 타입
