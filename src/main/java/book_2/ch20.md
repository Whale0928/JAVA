## java.lang 패키지

- import 할 필요 없이 바료 사용 할 수 있는 필수적인 Util들을 가지고 있는 클래스

java.lang에서 제공하는 기능
- 언어 관련 기본
- 문자열 관련
- 기본자료형  , 숫자 관련
- 쓰레드 관련
- 예외 관련
- 런타임 관련
