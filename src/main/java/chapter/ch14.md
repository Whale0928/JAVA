## Exception


- Try + Catch
- 무조건 한번은 수행하는 finally


- try 다음에 오는 catch는 1개 이상 일 수 있다.
- 먼저 선언한 catch 블록의 예외 클래스가 다음 선언한 요소의 부모에 속하면 다음 자식으로 포함된 예외 클래스는 절대 수행되지 않는다.
- 하나의 try에서 예외가 발생하면 위에서부터 순차적으로 catch 블록을 탐색한다.
- 예외에 일치하는 블록이 없으면 해당 쓰레드는 끝난다.
